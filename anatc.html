Updating the game

Sonic Forever 1.5.1 is set up differently to previous releases. Set the game up in a new folder as described above, then copy across SData.bin and settings.ini from your current game folder. Future updates can then be copied straight over.

Windows

1. Download from the below link.

2.Unzip the download

3. Add a Data.rsdk.xmf file from the Sonic the Hedgehog Classic app into the resulting folder.*

4. The game is now ready. Launch SonicForever.exe to play.

Android and switch beta

Sonic Forever is available on Android and Nintendo Switch. Install and run the executable so it can create a folder at RSDK/s1f in your filesystem. It may ask for permission to access files in order to do this. Then add the included mod folder and a data.rsdk file from Sonic 1 Classic into the s1f folder. This is still a work in progress, so you may encounter issues, and feedback would be very appreciated.


Note: The Switch beta will only work on modded/hacked Switches.
